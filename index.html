API Usage 
        # Get note content (JSON):
        curl http://notes.fleming.ai/api/notes/YOUR_NOTE_ID

        # Get note content (plain text):
        curl -H "Accept: text/plain" http://notes.fleming.ai/api/notes/YOUR_NOTE_ID

        # Save note content:
        curl -X POST -H "Content-Type: application/json" \
            -d "{\"content\":\"Your text here\"}" \
            http://notes.fleming.ai/api/notes/YOUR_NOTE_ID
